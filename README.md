# Asteroids
Space shooter video game inspired by Atari's Asteroids

![](https://github.com/YuviTz1/Asteroids/blob/main/Asteroids.gif)

## Dependencies
* GLFW
* glew
* glm
* CMake

## Build Instructions
After installing all the dependencies, from the project root directory
```
cd src
mkdir build
cd build
cmake ..
make
./Asteroids
```
